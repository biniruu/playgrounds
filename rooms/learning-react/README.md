# Play with React.js

## 설치 모듈

```bash
## PostCSS

$ yarn add -D postcss-cli postcss-url

## React

$ yarn add react-router-dom
$ yarn add -D @types/react-test-renderer react-test-renderer


## Sass loader

$ yarn add -D sass sass-loader


## Tailwind css

$ yarn add -D prettier-plugin-tailwindcss tailwindcss
```
